# mathandcoffe — Matematik Özel Ders Web Sitesi

Flutter 3 tabanlı mathandcoffe tanıtım sitesi; TYT/AYT, lise ve ortaokul matematik odaklı içerik sunan tek sayfa kaydırmalı bir web uygulamasıdır.

## Kurulum ve Çalıştırma

1. Gerekli paketleri indirin:
   ```bash
   flutter pub get
   ```
2. Uygulamayı Chrome üzerinde çalıştırın:
   ```bash
   flutter run -d chrome
   ```
3. Web sürümünü üretmek için:
   ```bash
   flutter build web
   ```

## Notlar
- Proje Material 3 teması, Google Fonts ve smooth scroll davranışı içerir.
- WhatsApp iletişim butonu `url_launcher` aracılığıyla yeni sekmede açılır.
